# MagicAI philosophy

MagicAI is built around a simple separation:

> Sources establish facts. Code enforces contracts. Models explain and explore.

## Source-grounded answers

The project retrieves current local evidence instead of relying on model memory. Oracle and rules may change; a trained memory can become stale.

## Judge authority

The Judge controls factual sources, provenance, validation, and legality. The Tactician may challenge a conclusion, but evidence decides the disagreement.

## Strategic autonomy

The Tactician should be persistent and competent. It may request repeated checks, compare alternatives, inspect interruption points, and declare missing capabilities. It should not accept the first plausible package merely because it can produce an answer.

## Generic fixes

A failing Young Wolf question should improve Undying, sacrifice, follow-up context, and validation generally—not add a special Young Wolf condition.

## Safe uncertainty

An explicit limitation is better than a confident contradiction of Oracle or the rules.
